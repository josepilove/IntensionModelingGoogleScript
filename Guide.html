How to use

This spreadsheet allows a workshop facilitator to collect, tabulate, and visualize the results from an InTension Modeling Workshop.
With continuums in-hand, it should only take a couple minutes to get this spreadsheet prepared for your workshop and worksheets ready to print from OmniGraffle.

The application was built with the following general workflow in mind:

  1. Create set of continuums
  2. Facilitate program workshop
    • Review tensions
    • Vote where we are today
    • Review today positions & edit/refine continuums
    • Vote where we want to be in the future
    • Review change in today v. future
    • Discuss
  3. Export results for archival purposes

About Settings Tab

This tab is used for setting up the sheet for your InTension Modeling workshop.

1: Input the number of Continuums for this workshop and click “Set Up Continuums”. This will create the right amount of blanks to input your contiuums values. “Terms” are required for each end, but “Descriptions” are optional.

2: Input the number of participants and click "Begin Inputting Votes". This is the last thing you need to do before the sheet is ready to collect votes.



About Worksheet Generation

Participant worksheets can be generated in conjunction with OmniGraffle. Once all the continuums values have been filled in on the Settings tab, click “Export Worksheets” and follow the instructions.

At this time, importing to OmniGraffle is the only way to generate worksheets automagically.



About Vote Collecting

For each continuum, a column is added and for each participant a row is added. If at any time you need more participant rows, adjust the number from the Settings tab. Nothing bad happens if you have empty/unused participant rows.

On the Vote Collection: Today tab, both the “Name” and the continuum value columns are editable. On the Vote Collection: Future tab, the “Name” column values are copied from the Vote Collection: Today tab.

We’ve found it easier to not have to worry about inputting negative numbers, so votes are not put in as -5 to 5. First votes are translated to positive numbers and inputted as 0 to 10.
(Translation is also included on the generated worksheets)



About Alignment Analysis

This sheet collects each participant’s votes and compares them to the aggregate of all participants.

Today Average (Alignment Analysis!F3) - Average of all the Today votes for a continuum.

Future Average (Alignment Analysis!F4) - Average of all the Future votes for a continuum.

Change Average (Alignment Analysis!F5) - The average difference between where the group voted for being today vs. where they voted for wanting to be in the future for a continuum.

Red = movement left;

Blue = movement right;
Saturation = amount of movement;

Amount of individual misalignment (Alignment Analysis!F6) - A value getting at the amount of discord amongst the population's votes. Takes into account both the direction of change as well as the amount of change. Lower numbers (green) indicate more-aligned continuums.

Today (Alignment Analysis!F10) - Individual's votes for where things are today for a continuum.

Future (Alignment Analysis!F11) - Individual's votes for where things should be in the future for a continuum.

Change (Alignment Analysis!F12) - The change between where an individual voted for being today vs. where they voted for wanting to be in the future for a continuum. Red = a left-ward move; Blue = a right-ward move; Darker color = larger movement;

Alignment with the average (Alignment Analysis!F13) - This individual's alignment with the group's average for a continuum.

Number of Misalignments (Alignment Analysis!A1) - This is the number of continuums for which this individual voted significantly in the opposite direction of the group average. This is helpful for identifying individual outliers and making sure that key stakeholders aren't in silent opposition.



About Visualizations

For each continuum, a visualization can be generated with either just Today’s votes or both Today and Future votes populated.

Each vote is represented as a “chicklet” along the continuum, either above (today) or below (future) the continuum line.

The average position of a continuum is shown as a dark chicklet along the continuum line. When the average chicklet overlaps with a vote chicklet no text is present, when the average chicklet is not on top of a vote chicklet, “Average Only” is in the average chicklet to indicate that there is no vote in that position.



About Output Generation

The visualizations can all be exported at once into omnigraffle for further manipulation and archiving.
Click “InTension” > “Export Output” and follow the instructions.
